A project created to simulate an AI agent playing the 3-player Kuhn Poker variant.

Project uses skeleton provided by Annual Computer Poker Competition: 
http://www.computerpokercompetition.org/index.php/competitions/participants/103-participants-2014?showall=&start=4

Project uses a strategy based on Nash Equilibria for 3-player Kuhn Poker and Fictitious Play algortihm to attempt and balance utility between opponents if no equilibrium play is beneficial.
